# C# Essentials Refresher

A compact project to brush up on core **C# language fundamentals**—perfect for revisiting syntax, concepts, and best practices.

## 📌 Overview

This repository covers essential C# topics with simple examples and exercises. It’s designed for:

* Developers returning to C#
* Beginners reinforcing fundamentals
* Interview preparation

## 🧠 Topics Covered

* Basic syntax and structure
* Variables and data types
* Control flow (`if`, `switch`, loops)
* Methods and parameters
* Object-Oriented Programming (OOP)

  * Classes & objects
  * Inheritance
  * Polymorphism
  * Encapsulation
* Collections (`List`, `Dictionary`, arrays)
* Exception handling
* LINQ basics
* File I/O (optional)

## 🛠️ Requirements

* .NET SDK (6.0 or later recommended)
* Any C# IDE:

  * Visual Studio
  * VS Code
  * Rider

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/csharp-essentials.git

# Navigate into the project
cd csharp-essentials

# Run the project
dotnet run
```

## 📂 Project Structure

```
/Basics          # Syntax, variables, control flow
/OOP             # Classes, inheritance, polymorphism
/Collections     # Lists, dictionaries, arrays
/LINQ            # Query examples
/Exceptions      # Error handling
/Exercises       # Practice problems
```

## ✍️ How to Use

* Read through each folder sequentially
* Run examples and modify them
* Try exercises before checking solutions
